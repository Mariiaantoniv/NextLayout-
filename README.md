# Next Layout
Modern landing page for an online store offering tech devices, complete with contact information and a quick feedback form.

## Project Description
This is a modern landing page for an online store offering technological devices, featuring key product sections, contact information, and a quick feedback form. Users can explore the store’s main advantages, browse informational blocks, and easily get in touch by filling out the interactive form. The layout is designed in a minimalist style with a focus on clean structure and user-friendly interaction.

## Technical Requirements
- Modern web browser — Chrome, Firefox, Safari, Edge or any browser supporting HTML5 and CSS3.
- Internet connection — only if external assets (fonts, icons, images) are loaded.
- Code editor (optional) — such as Visual Studio Code, if you want to explore or modify the source code.

## Usage
After starting the project, it will be available at http://localhost:8080. You can use this URL to preview the landing page and ensure it displays correctly. You can also make changes to the project files and see updates in real-time.

## Features
- Responsive Layout — optimized for mobile, tablet, and desktop screens
- Interactive Mobile Menu — smooth open/close animations for improved navigation
- Recommended Products Section — clean product cards with images, descriptions, and pricing
- Category Grid — adaptive image layout showcasing product categories
- Modular SCSS Structure — variables, mixins, and partials for scalable styling
- Optimized Assets — compressed images and efficient bundling for fast load times
- Clean UI/UX — minimalistic design with strong visual hierarchy and consistent spacing

## Technologies Used
This project was built using the following technologies:

- HTML5 — used to structure the content of the landing page with semantic and accessible markup.
- CSS3 — responsible for styling, layout, and responsive design across different screen sizes.
-Sass — a CSS preprocessor that enables cleaner and more maintainable styling through variables, nesting, mixins, and modular structure.
