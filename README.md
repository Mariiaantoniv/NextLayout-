# Base layout template with Gulp, SCSS and Stylelint
1. B&Q — Audio Store Landing Page
A modern, responsive landing page created for a premium audio brand.
The project focuses on clean semantic HTML, modular SCSS architecture, and a visually appealing layout inspired by high‑end product websites. It includes interactive UI elements, adaptive image grids, and a mobile‑friendly navigation experience.
The goal is to deliver a polished, user‑centric interface that highlights product categories and featured items through elegant design and smooth interactions.
2. - [DEMO LINK](https://Mariiaantoniv.github.io/NextLayout-/)
3. Technologies Used
- HTML5 — provides a clean, semantic structure that improves accessibility, SEO, and maintainability of the layout.

- SCSS — enables modular styling with variables, mixins, and partials, making the codebase scalable and easy to extend.

- JavaScript — adds interactivity to the interface, including mobile navigation behavior and dynamic UI elements.

- Parcel — a fast, zero‑configuration bundler that handles development server, hot reload, asset optimization, and production builds.

4. Features
- Responsive Layout — optimized for mobile, tablet, and desktop screens
- Interactive Mobile Menu — smooth open/close animations for improved navigation
- Recommended Products Section — clean product cards with images, descriptions, and pricing
- Category Grid — adaptive image layout showcasing product categories
- Modular SCSS Structure — variables, mixins, and partials for scalable styling
- Optimized Assets — compressed images and efficient bundling for fast load times
- Clean UI/UX — minimalistic design with strong visual hierarchy and consistent spacing
5. Responsiveness
The layout is fully responsive and adapts seamlessly across devices:
Mobile‑first approach
Optimized breakpoints for phones, tablets, and desktops
Flexible grid system and adaptive images
Touch‑friendly navigation and interactive elements
